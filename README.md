# Smoker

A management system designed for small businesses, developed in Java.

##  About the Project
This project was built to help small companies manage their inventory/sales/customers efficiently. It focus on a lightweight architecture and ease of use for offline environments.

##  Technologies
* **Java** (Core logic)
* **NetBeans IDE** (Development environment)
* SQLite


##  How to run
1. Clone the repository.
2. Open the project in **NetBeans** or **VsCode**.
3. Run the `Main` class.
